## Overview

A modern, professional sponsorship pitch deck for Crew Development Institute (CDI), designed to match the clean, Africa-focused nonprofit aesthetic of CBA for Africa.

## Design Elements

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| CDI Gold | `#C9A227` | Primary accent, highlights |
| CDI Gold Light | `#E5C558` | Secondary accent |
| Dark Background | `#0A0A0A` | Primary background |
| Gray Dark | `#1A1A1A` | Card backgrounds |
| Gray | `#2A2A2A` | Borders |
| Gray Light | `#8A8A8A` | Body text |
| White | `#FFFFFF` | Headlines |
| Ghana Red | `#CE1126` | Ghana stripe |
| Ghana Yellow | `#FCD116` | Ghana stripe |
| Ghana Green | `#006B3F` | Ghana stripe |

### Typography
- **Headlines:** Space Grotesk (Bold, 700)
- **Body:** Inter (Regular 400, Medium 500)
- Clean hierarchy with varying sizes

### Layout
- Full-viewport slides with smooth scrolling
- Card-based content sections
- Navigation dots on right side
- Responsive down to mobile

## Slides Included

1. **Cover** — CDI logo, mission tagline, Ghana colors
2. **The Opportunity** — Problem statistics (85% imported crews, $2B+ leaving continent)
3. **What CDI Does** — Three pillars (Learning, Advisory, Production)
4. **Impact & Vision** — Training metrics and tracks
5. **Partnership Tiers** — Bronze ($2,500) / Silver ($5,000) / Gold ($10,000) / Platinum ($25,000+)
6. **Leadership Team** — Erica Hayes, Keith Bernard, Blake Brady + Illumin8 partner
7. **Call to Action** — Contact info and partner link

## How to Use

### Web Viewing
Open `index.html` in any modern browser. Navigation dots allow quick jumping between slides.

### PDF Export
1. Open in Chrome/Safari
2. Press `Cmd/Ctrl + P` to print
3. Select "Save as PDF"
4. The print styles will automatically activate for clean PDF output

### Presenting
- Each slide fills the viewport
- Scroll or use navigation dots
- Works on both desktop and mobile
